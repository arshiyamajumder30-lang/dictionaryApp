# 📖 English Dictionary App

A simple English Dictionary web application built using HTML, CSS, and JavaScript. Users can search for English words and get their meanings, definitions, and pronunciation audio.

## Features

- Search for any English word
- View meanings and definitions
- Listen to pronunciation audio
- Clean and responsive UI
- Real-time data using Dictionary API

## Tech Stack

- HTML
- CSS
- JavaScript
- Dictionary API
